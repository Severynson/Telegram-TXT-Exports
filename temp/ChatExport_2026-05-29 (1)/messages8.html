Привет
Посмотрели какие метрики можем делать для пиклбола.
1. Service Efficiency :
- Процент успешных подач
- Количество "эйсов"
- Процент ошибок при подаче
2. Ball Placement :
- Общее количество ударов
- Точность ударов в различные зоны поля
- Количество ударов, приведших к победе в розыгрыше
- Процент успешных "Median Drives" и "Median Serves"
3. Net Play :
- Количество успешных виннеров (ударов, после которых мяч не возвращен противником) в сетке.
- Процент успешных "Dinking" розыгрышей (мягких ударов над сеткой)
4. Error Metrics :
- Количество ошибок при ударах
- Процент ошибок относительно общего числа ударов
- Количество двойных ошибок при подачах
5. Average Rally Time
- Время, которое требуется для завершения каждого розыгрыша
6. High Ball Handling Efficiency
- Процент успешных ударов при обработке высоких мячей
