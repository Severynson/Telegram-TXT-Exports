# Requirements:

- the app has to be implemented for all OSs:

* MacOS
* Linux
* Windows

- The app is focused to be conveniently distributed to non-technical users and being convenient for them. There always has to be a folder structure - so every user could quickly find their OS-compatible version and run it by double clicking, without having to make sure they have python installed:

* app/macos
* app/linux
* app/windows
